# MIPS_Basics
Basic MIPS Concepts

#In order to run MIPS Code you need 2 things
1.) Basic text editor
2.) QTSpim

#Steps to run code
1.) After editing the given assembly code in text editor save the file with a '.asm' suffix
2.) Open up QTSpim
3.) Select File
    -Reinitialize and LoadFile
    -Select the file you wish to run
    -Select play button
4.) View Console
